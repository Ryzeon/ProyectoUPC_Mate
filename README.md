# Installation
```bash
git clone https://github.com/Ryzeon/ProyectoUPC_Mate.git
cd ProyectoUPC_Mate 
# If you are in linux
./mvnw clean javafx:run
# If you are in windows
mvnw.cmd clean javafx:run
```
